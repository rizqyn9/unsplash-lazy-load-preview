# Unsplash lazy load preview

### Run Palindrome

```
yarn palindrome
```

<br/>

### Development

```
yarn install
```

```
yarn start
```

<br/>

### Build

```
yarn build
```

<br/>

### Run production

```
yarn start
```

<br/>

### Tools

- [React Query](https://tanstack.com/query/v4)
- [Tailwindcss](https://tailwindcss.com/)
- [Jotai](https://jotai.org/)
- [Typescript](https://www.typescriptlang.org/)
